# Skills

Personal collection of AI agent skills for Claude Code.

## Skills

- [monster-template-generator](./monster-template-generator/) — 为"小怪打卡队"App 生成可导入的图鉴模板 ZIP 包

## Install

```bash
npx skills add yangyi/skills
```

Or install a specific skill:

```bash
npx skills add yangyi/skills/monster-template-generator
```
